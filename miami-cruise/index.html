🌆 マイアミ 注意事項

🚨 治安・危険エリア

  • 北部エリア（リトル・ハイチ、リバティ・シティ）は近づかない
  • 夜間の一人歩きは避ける
  • スリ・置き引きに注意（空港、ホテルロビー、ビーチ周辺）
  • 派手な格好・高価なアクセサリーは目立つので注意

🚗 移動・駐車

  • 移動は Uber / Lyft を使うのが安全で便利
  • タクシーより安く、安全に確認できる
  • レンタカーの場合、車内に荷物を絶対に置かない（窓を割られる）
  • 駐車場代は高額（WBC会場周辺は特に）

💵 チップの目安

  • レストラン: 合計金額の18〜20%
  • バー: $1〜2 / 1ドリンク
  • タクシー・Uber: 10〜15%
  • ホテルポーター: $2〜3 / 荷物1個
  • ハウスキーピング: $3〜5 / 泊
  • 小額紙幣（$1, $5）を多めに用意しておくと便利

🌤️ 気候・服装

  • 3月のマイアミは 25〜30℃、湿度高め
  • 突然のスコールあり → 折りたたみ傘を携帯
  • 冷房が強い室内向けに薄手の羽織りものを
  • 夜はやや涼しくなる場合あり（試合観戦時）

✈️ 空港→ホテル アクセス

🚌 エアポートマリオット 無料シャトル

マイアミ国際空港（MIA）から約1.9マイル（約3km）の距離。ホテルが無料シャトルバスを運行中。到着ロビーのシャトル乗り場から乗車可（所要約4〜5分）。念のため到着前にホテルへ確認しておくと安心。

🚗 その他の空港→ホテル移動手段

  • タクシー: 約$11〜14（4分）
  • Uber / Lyft: 同程度の料金
  • 公共バス（Airport Flyer）: $2.25（安いが荷物多い時は不向き）
  • 深夜着の場合はUberが確実

🏨 ホテル→クルーズターミナルへの移動（3/18）

  • 乗船は17:00（荷物預けは9:00〜）
  • ホテルからターミナルVまでUberで約15〜20分
  • Marriottがクルーズ用シャトルサービスを提供している場合あり（要確認）
  • 荷物が多いので早めに出発推奨

💳 その他の注意事項

📱 通信・Wi-Fi

  • 海外SIMまたは国際ローミングを事前に準備
  • Uberの利用にはスマホとカードが必須
  • Google マップを事前にダウンロード（オフライン対応）

💊 健康・緊急時

🚢 クルーズ（Scarlet Lady）注意事項

📋 乗船前に必ずやること

📱 Virgin Voyagesアプリ

  • 必須: "Virgin Voyages" アプリを事前インストール
  • Terminal Arrival Time（乗船時間）をアプリで選択する
  • 出航24時間前に健康チェックフォームをアプリで回答
  • 船内はアプリで全て管理（レストラン予約・スパ予約等）

📄 必要書類

  • パスポート必須（帰国後6ヶ月以上有効なもの）
  • クルーズ予約確認書
  • 乗船時はパスポートとアプリを手元に
  • 荷物預けは午前9時から可能
  • ターミナルV（Terminal V）に向かう

🚢 船内のルール・知っておくこと

✅ チップ・支払い

  • チップ不要（Virgin Voyagesはチップ込み）
  • 船内の支払いはクレジットカードのみ
  • チェックイン時にカード登録が必要
  • 飲食は多くのレストランが含まれている

🚫 持ち込み禁止物

  • アルコール飲料（船外からの持ち込み不可）
  • 違法薬物・CBD製品
  • 銃器・爆発物・刃物
  • 一回使い捨てプラスチック
  • ドローン

👗 ドレスコード・イベント

  • ドレスコードなし（カジュアルでOK）
  • Scarlet Night: 赤い服を必ず持参！（全員参加が盛り上がる）
  • PJパーティー: パジャマを持参
  • イベントはアプリで事前確認・予約

🌊 船酔い対策

  • 3/19は終日航海 → 酔いやすい人は対策を
  • 酔い止め薬（市販）を日本から持参
  • 船内にも販売あるが割高
  • 船の中央・低い階の部屋が揺れにくい

🧴 日焼け止めについて

船内・寄港地ではリーフセーフ（環境に優しい）日焼け止めのみ使用・販売。珊瑚礁を守るため、ケミカル成分（オキシベンゾン等）を含むものは寄港地では使用不可の場合あり。日本からミネラル（ノンケミカル）タイプを持参推奨。

💳 別途料金が必要なもの

  • 🍺 アルコール飲料 → Bar Tabで事前購入が割安
  • ☕ スペシャルコーヒー（ラテ・カプチーノ等）・生ジュース・スムージー
  • 🌐 上位Wi-Fi（ストリーミング対応：Essential/Premiumファア）
  • ♨️ サーマルスイート（スパ）: $59〜79 / 3時間
  • 💆 スパトリートメント（マッサージ等）
  • 🏝️ Shore Things（オプショナルツアー）
  • 🧺 ランドリーサービス

🍺 Bar Tab（バータブ）システムについて

Virgin Voyagesはオールインクルーシブの飲み放題パッケージがありません。代わりにBar Tabという事前チャージ制を採用。

  • 事前にクレジットを購入するとボーナスが付く（例: $300購入→$350分使える）
  • アプリまたは船内バーで残高確認可能
  • 使い切れなかった場合は返金あり
  • 飲みたい人は乗船前に購入しておくとお得

🍽️ レストラン予約 攻略法

⏰ 予約開始タイミング

  • 予約はアプリで Discover → Ship Spaces → Eateries から
  • Baseファア: 出航15日前から予約開始
  • Essentialファア: 出航45日前から
  • Premiumファア: 出航60日前から
  • 予約開始は深夜0時（東部時間）、人気店はすぐ埋まる

🎯 予約のコツ

  • 1日1予約まで（ルール）
  • 全日程を予約しすぎない→ウォークインやGalley、深夜ピザも楽しい
  • 予約が埋まっていてもキャンセルが出るので諦めず確認
  • 船内でも予約枠が追加で開放される
  • アプリが重い場合はウェブサイト（virginvoyages.com）からも予約可能

  • 水深が浅くシュノーケリングで十分楽しめる
  • アトランティス伝説の石積み「ビミニロード」も近くにある
  • 船内に無数の魚・珊瑚が生息、写真映え◎

🎣 ④ スポーツフィッシング

  • ビミニは「世界のスポーツフィッシングの聖地」
  • ヘミングウェイが愛した漁場
  • マグロ・カジキ・ワフー等の大物が狙える
  • 半日チャーター: 約$400〜800 / 艇（グループ割れ）
  • 要事前予約 / Big Game Club Biminíで手配可

🐟 ⑤ エイの餌付け体験

  • 浅瀬で野生のエイに直接餌付けができる
  • ガイド付きで安全に体験可能
  • 泳げなくても大丈夫（水深が浅い）
  • サメ餌付け体験とセットのツアーもあり

🏖️ ⑥ Resort World Biminiでのんびり

📸 見るべきスポット

🪨 ビミニロード（アトランティスの道）

  • 海底に並ぶ謎の石積み（アトランティス伝説の遺跡とも言われる）
  • ダイビング・シュノーケリングで見学可能
  • 世界中のダイバーが訪れる神秘的なスポット

🐬 イルカの家博物館（Dolphin House Museum）

  • ローカルアーティストが30年かけて建設した奇妙な建物
  • 貝殻・珊瑚・シーグラスで装飾された外壁が圧巻
  • 地元の歴史・文化がわかるユニークな場所

🏖️ Radio Beach

  • ビミニ一の美しい白砂ビーチ
  • 透明度の高いエメラルドグリーンの海
  • 泳ぎ・シュノーケリングに最適
  • 港から徒歩圏内
