🌆 マイアミ 注意事項

🚨 治安・危険エリア

  • 北部エリア（リトル・ハイチ、リバティ・シティ）は近づかない
  • 夜間の一人歩きは避ける
  • スリ・置き引きに注意（空港、ホテルロビー、ビーチ周辺）
  • 派手な格好・高価なアクセサリーは目立つので注意

🚗 移動・駐車

  • 移動は Uber / Lyft を使うのが安全で便利
  • タクシーより安く、安全に確認できる
  • レンタカーの場合、車内に荷物を絶対に置かない（窓を割られる）
  • 駐車場代は高額（WBC会場周辺は特に）

💵 チップの目安

  • レストラン: 合計金額の18〜20%
  • バー: $1〜2 / 1ドリンク
  • タクシー・Uber: 10〜15%
  • ホテルポーター: $2〜3 / 荷物1個
  • ハウスキーピング: $3〜5 / 泊
  • 小額紙幣（$1, $5）を多めに用意しておくと便利

🌤️ 気候・服装

  • 3月のマイアミは 25〜30℃、湿度高め
  • 突然のスコールあり → 折りたたみ傘を携帯
  • 冷房が強い室内向けに薄手の羽織りものを
  • 夜はやや涼しくなる場合あり（試合観戦時）

💳 その他の注意事項

📱 通信・Wi-Fi

  • 海外SIMまたは国際ローミングを事前に準備
  • Uberの利用にはスマホとカードが必須
  • Google マップを事前にダウンロード（オフライン対応）

💊 健康・緊急時

  • 海外旅行保険に必ず加入（医療費が非常に高額）
  • 常備薬は多めに持参
  • 緊急: 911 / 在マイアミ日本国総領事館: +1-305-530-9090
⚾ WBC観戦（loanDepot Park）注意事項
🎟️ バッグポリシー: クリアバッグのみ入場可（透明のバッグ限定）。通常のバッグは持ち込み不可です。事前に準備必須！

🚫 持ち込み禁止物

  • ガラス瓶・金属缶（水筒も不可）
  • アルコール・清涼飲料水・スポーツドリンク
  • タバコ製品（会場内禁煙）
  • ハードサイドのクーラー・保冷バッグ
  • 3'×5'を超える横断幕・旗
  • ドローン、レーザーポインター、花火
  • 武器・おもちゃの武器
  • 外部からの食べ物（制限あり）

✅ 持ち込めるもの

  • 未開封のペットボトル水 20oz（約600ml）以下 1本
  • クリアバッグ 16"×16"×8"以下
  • 小型クラッチ 4.5"×6.5"以下（非透明OK）
  • 医療・育児用品（スタッフに申告）
  • カメラ（フラッシュなし、業務用レンズ不可）

⏰ 時間・アクセス

  • 試合開始は21:00または20:00→ 2〜3時間前に会場着を推奨
  • セキュリティチェックに時間がかかる
  • Uber/Lyftで行くのが便利（駐車場は高額）
  • 帰りはUberが捕まりにくいので早めに呼ぶ

🍔 会場内で買えるもの

  • 飲食は全て会場内で購入可能
  • ビール・アルコールあり（会場内購入）
  • クレジットカード払い対応
  • 現金より カード払いが主流

🚢 クルーズ（Scarlet Lady）注意事項

📋 乗船前に必ずやること

📱 Virgin Voyagesアプリ

  • 必須: "Virgin Voyages" アプリを事前インストール
  • Terminal Arrival Time（乗船時間）をアプリで選択する
  • 出航24時間前に健康チェックフォームをアプリで回答
  • 船内はアプリで全て管理（レストラン予約・スパ予約等）

📄 必要書類

  • パスポート必須（帰国後6ヶ月以上有効なもの）
  • クルーズ予約確認書
  • 乗船時はパスポートとアプリを手元に
  • 荷物預けは午前9時から可能
  • ターミナルV（Terminal V）に向かう

🚢 船内のルール・知っておくこと

✅ チップ・支払い

  • チップ不要（Virgin Voyagesはチップ込み）
  • 船内の支払いはクレジットカードのみ
  • チェックイン時にカード登録が必要
  • 飲食は多くのレストランが含まれている

🚫 持ち込み禁止物

  • アルコール飲料（船外からの持ち込み不可）
  • 違法薬物・CBD製品
  • 銃器・爆発物・刃物
  • 一回使い捨てプラスチック
  • ドローン

👗 ドレスコード・イベント

  • ドレスコードなし（カジュアルでOK）
  • Scarlet Night: 赤い服を必ず持参！（全員参加が盛り上がる）
  • PJパーティー: パジャマを持参
  • イベントはアプリで事前確認・予約

🌊 船酔い対策

  • 3/19は終日航海 → 酔いやすい人は対策を
  • 酔い止め薬（市販）を日本から持参
  • 船内にも販売あるが割高
  • 船の中央・低い階の部屋が揺れにくい

🧴 日焼け止めについて

船内・寄港地ではリーフセーフ（環境に優しい）日焼け止めのみ使用・販売。珊瑚礁を守るため、ケミカル成分（オキシベンゾン等）を含むものは寄港地では使用不可の場合あり。日本からミネラル（ノンケミカル）タイプを持参推奨。
